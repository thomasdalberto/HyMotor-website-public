5. Data Subject Rights

Under the GDPR the user has the right to:

  1. Access personal data concerning them (Art. 15).
  2. Rectify inaccurate data (Art. 16).
  3. Erase data (Art. 17 — right to be forgotten).
  4. Restrict processing (Art. 18).
  5. Data portability (Art. 20).
  6. Object to processing (Art. 21).
  7. Withdraw consent at any time (Art. 7(3)), without affecting the lawfulness of prior processing.
  8. Lodge a complaint with the supervisory authority.

To exercise these rights, contact the Controller at: privacy@hymotor.it.

Cookie Policy

Cookies are small text files (letters and/or numbers) that allow a web server to store information in the browser to be reused during the same visit (session cookies) or later (persistent cookies).

This site uses the following types of cookies and local storage:

A. Technical Cookies (Always Active)

These are necessary for correct functioning and provision of the service. They do not require user consent.

Name / Storage Category Purpose Duration
analyticsConsent (Local Storage) Technical / Functional Stores the user’s cookie consent status (Accepted/Rejected). Persistent
theme (Local Storage) Functional Stores the display theme preference (dark/light). Does not contain personal data. Persistent

B. Third-Party Analytical Cookies (Require Consent)

These cookies are used for statistical analysis and are set only after the user has given consent via the Cookie Banner.

Cookie name Provider Purpose Duration
_ga Google Analytics 4 Distinguish users for analytics purposes. 2 years
_ga_<ContainerID> Google Analytics 4 Maintain session state. 2 years

Consent for analytical cookies (GA4) is managed via the Cookie Banner. If the user does not give consent, analytical cookies will not be activated.
